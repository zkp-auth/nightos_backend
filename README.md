# NightOps Backend

NightOps is a backend platform designed to manage DJ bookings and operational workflows for nightlife venues.

The goal is to replace manual processes (paper, spreadsheets, WhatsApp follow-ups) with a structured, reliable, and scalable system.

---

## 🎯 Purpose

- Simplify DJ booking management
- Track confirmations efficiently
- Reduce delays and errors (double bookings, missed confirmations)
- Provide a clear operational view for managers

---

## ⚙️ Tech Stack

- Python 3
- Django
- Django REST Framework
- PostgreSQL (Docker)
- Docker Compose
